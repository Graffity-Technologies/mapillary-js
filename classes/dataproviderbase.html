Class DataProviderBase

classdesc

Base class to extend if implementing a data provider class.

example
class MyDataProvider extends Mapillary.API.DataProviderBase {
     constructor() {
         super(new Mapillary.API.S2GeometryProvider());
     }
     ...
}
