Class KeyboardComponent

class

KeyboardComponent

classdesc

Component for keyboard event handling.

To retrive and use the keyboard component

example
var viewer = new Mapillary.Viewer(
    "<element-id>",
    "<client-id>",
    "<my key>");

var keyboardComponent = viewer.getComponent("keyboard");
