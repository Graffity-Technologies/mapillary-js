Class PointsGeometry

classdesc

Represents a point set in the 2D basic image coordinate system.

example
var points = [[0.5, 0.3], [0.7, 0.3], [0.6, 0.5]];
var pointsGeometry = new Mapillary.TagComponent.PointsGeometry(points);
