isFallbackSupported

  • isFallbackSupported(): boolean
    • Defined in src/Support.ts:29

    Test whether the current browser supports the fallback functionality of MapillaryJS.

    description

    The fallback functionality does not include WebGL rendering, only 2D canvas rendering.

    example

    var fallbackSupported = Mapillary.isFallbackSupported();

    Returns boolean
